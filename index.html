Introduction to HTML

HTML (HyperText Markup Language) is the foundational language of the web. It provides the structure for web pages through a system of tags and attributes. This guide will take you from complete beginner to advanced HTML developer.

What You'll Learn

  • HTML document structure and syntax
  • Semantic HTML for accessibility and SEO
  • Forms, tables, and multimedia elements
  • Advanced APIs and web components
  • Best practices and modern techniques

Why Learn HTML?

  • Essential for all web development
  • Improves accessibility for all users
  • Critical for search engine optimization
  • Foundation for CSS and JavaScript
  • Universal browser support

Beginner Level: HTML Fundamentals

Basic Document Structure

Every HTML document has a standard structure:

<!DOCTYPE html> <!-- Document type declaration --> <html> <!-- Root element --> <head> <!-- Metadata container -->   <title>Page Title</title> <!-- Browser tab title --> </head> <body> <!-- Visible content -->   <h1>My First Heading</h1>   <p>My first paragraph.</p> </body> </html>

Core Elements

Text Elements

  • Headings: <h1> to <h6>
  • Paragraphs: <p>
  • Text formatting: <strong>, <em>
  • Lists: <ul>, <ol>, <li>
  • Links: <a href="...">

Media Elements

  • Images: <img src="..." alt="...">
  • Audio: <audio controls>
  • Video: <video controls>
  • Embed: <embed>, <iframe>

Structural Elements

  • Divisions: <div>
  • Spans: <span>
  • Line breaks: <br>
  • Horizontal rules: <hr>

Intermediate Level: Structured HTML

Semantic HTML

Semantic elements clearly describe their meaning to both browser and developer:

Semantic Tag Non-Semantic Equivalent Purpose
<header> <div id="header"> Introductory content or navigation
<nav> <div class="nav"> Navigation links
<main> <div id="main"> Primary content of the document
<section> <div class="section"> Thematic grouping of content
<article> <div class="article"> Self-contained composition
<aside> <div class="sidebar"> Content indirectly related to main content
<footer> <div id="footer"> Footer for its nearest section

Forms and Tables

Form Elements

  • <form> - Container for form elements
  • <input> - Various input types
  • <textarea> - Multi-line text input
  • <select> - Dropdown list
  • <button> - Clickable button
  • <label> - Label for form controls

Input Types

  • text, password, email, url
  • number, range, date, time
  • checkbox, radio, file
  • submit, reset, button

Table Structure

<table>   <caption>Table Title</caption>   <thead>     <tr>       <th>Header 1</th>       <th>Header 2</th>     </tr>   </thead>   <tbody>     <tr>       <td>Data 1</td>       <td>Data 2</td>     </tr>   </tbody> </table>

Advanced Level: Modern APIs & Components

Web Components

Web Components are a set of web platform APIs that allow you to create new custom, reusable, encapsulated HTML tags.

Custom Elements

<!-- Define custom element --> <script> class UserCard extends HTMLElement {   constructor() {     super();     this.innerHTML = `       <div class="user-card">         <h2>${this.getAttribute('name')}</h2>         <p>${this.getAttribute('title')}</p>       </div>     `;   } } customElements.define('user-card', UserCard); </script>
<!-- Use custom element --> <user-card   name="Jane Smith"   title="Web Developer" ></user-card>

Shadow DOM

Provides encapsulation by separating the DOM of a component from the main document DOM:

const shadow = this.attachShadow({mode: 'open'}); const style = document.createElement('style'); style.textContent = `   .card {     padding: 20px;     border: 1px solid #ccc;   } `; shadow.appendChild(style); shadow.appendChild(cardContent);

Modern APIs

Geolocation API

navigator.geolocation.getCurrentPosition(   (position) => {     const lat = position.coords.latitude;     const lon = position.coords.longitude;     console.log(`Latitude: ${lat}, Longitude: ${lon}`);   },   (error) => {     console.error("Error getting location", error);   } );

Web Storage API

// Save data to localStorage localStorage.setItem('username', 'john_doe');
// Retrieve data const user = localStorage.getItem('username');
// Save session data sessionStorage.setItem('sessionToken', 'abc123');
